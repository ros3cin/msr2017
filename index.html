A Study on the Energy Consumption of Android App Development Approaches

Using code from TCLBG, Rosetta Code and F-droid we benchmark and measure the energy consumption and the performance of different Android development approaches to answer the following questions:

1. Is there a more energy-efficient app development approach?

For this question, we ran benchmarks on the following devices:

2. Can a hybrid approach to app development save energy?

For this question we used real apps from the F-droid online repository. For each one of them, we varied between the different ways of hybridizing a Android app, discussed in our paper.
