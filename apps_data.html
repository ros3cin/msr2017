Apps comparison

Using the three different development approaches available for Android - Java, NDK, Web - we compare the following apps, taken from F-droid repository:

  • Andof
  • Enigmandroid
  • TriRose

We define three different ways of using NDK or Javascript to do part of the job of a Android application:

  • Stepwise

    Executes the method once, several times, returning a single value each time.

  • Export

    Executes the method several times, returning a single value.

  • Batch

    Executes the method several times, returning clustered data.
